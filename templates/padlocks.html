How the score is calculated

The padlock score starts from a base score (0–5) determined by the total number of studies in the evidence base. One padlock is then deducted for each of the five secondary risk indicators rated H (High risk). The minimum score is 0 (or 1 if there are more than 9 studies).

Base (N studies) − 1 if % since 2000 = H − 1 if % randomised = H − 1 if % eco. valid = H − 1 if % indep. eval = H − 1 if % attrition = H = {{ data.New_padlock|int }} padlocks
